## Description

Final proyecto of Bootcamp de JavaScript en el Backend

## Installation

```bash
npm install
```

## Enviroment

```bash
# copy env
$ cp .env.example .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## docker compose

```bash
#start Docker
$ docker compose up
```

## Postman

```bash
#copy postman
$ cd docs
```

## swagger

```bash
#open swagger
$ http://localhost:3000/api
```
